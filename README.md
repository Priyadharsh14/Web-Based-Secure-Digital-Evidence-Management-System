# Digital Evidence Vault

A secure web-based system for managing, storing, and tracking digital evidence with strong encryption and audit capabilities.

---

## Overview

The Digital Evidence Vault is designed to ensure secure handling of digital evidence in a structured and tamper-proof manner.  
It allows investigators to upload evidence linked to cases and enables authorized personnel to access and download files securely.  

The system ensures:
- Confidentiality through encryption  
- Integrity through hashing  
- Accountability through audit logs  

---

## Features

- Role-Based Access Control (Investigator / Authority)  
- Case Creation and Management  
- Secure Evidence Upload (Encrypted Storage)  
- Controlled Evidence Download  
- AES Encryption and RSA Key Protection  
- SHA-256 Integrity Verification  
- Audit Logs with Chain-of-Custody Tracking  
- Search Functionality for Cases  
- User-Friendly Dashboard
